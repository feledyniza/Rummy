Get started with Sprite Atlas Analysis

Get started with using the Sprite Atlas Analysis feature of the 2D Tooling package.

The Sprite Analyzer window displays detailed information about the 2D texture assets contained in all sprite atlases within your project. This information is organized into six types of reports. This helps you to efficiently manage and optimize your 2D assets, as well as help you to quickly identify common issues via the different report views for specific information.

First install the package, then go to Window > Analysis > Sprite Atlas Analyzer to launch a new window for analyzing the sprite atlases in the project.

Using the Sprite Analyzer

After launching the Sprite Analyzer window, click the Analyze button to begin analyzing the contents of the sprite atlases in the project. The Analyzer window provides six different specific reports. Refer to Built-in reports for the different types of reports and information available in the analyzer.

Configuring the Analyzer's data source

To configure the Analyzer's data source settings, click the More (...) menu next to the Analyze button to open the Data Source Configuration panel. You can set the path for the data source that the analyzer by adding or removing different folders for analysis.
