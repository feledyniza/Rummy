Input nodes

Supply shaders with essential data such as constants, mesh attributes, gradients, matrices, deformation, PBR parameters, scene details, and texture sampling options.

Integer Defines a constant Float value in the shader using an Integer field.
Slider Defines a constant Float value in the shader using a Slider field.
Time Provides access to various Time parameters in the shader.

Mesh Deformation

Topic Description
Compute Deformation Node Passes compute deformed vertex data to a vertex shader. Only works with the Entities Graphics package.

Sprite Deformation

Topic Description
Sprite Skinning Node Applies Vertex Skinning on Sprites. Only works with the 2D Animation.
